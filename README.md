Color detection system using AI 

The Color Detection System using AI and OpenCV is an intelligent image processing project developed to identify and recognize colors from images in real time. The system uses the Python programming language along with the OpenCV library for image processing and a CSV-based RGB color dataset for accurate color matching.

In this project, an image is loaded into the system and the user can click on any part of the image to detect the color present at that pixel location. The program extracts the RGB (Red, Green, Blue) values of the selected pixel and compares them with the RGB values stored in the dataset. To find the closest matching color, the system uses the Manhattan Distance Formula, which calculates the difference between RGB values of the selected pixel and dataset colors. The color with the minimum distance is identified as the detected color.

The project provides a user-friendly graphical interface where the detected color name and its RGB values are displayed on the screen. This system demonstrates the practical implementation of Artificial Intelligence concepts, computer vision, and machine learning techniques in image analysis.

The main objective of this project is to simplify color identification and improve accuracy in detecting colors from digital images. It can be used in various applications such as graphic design, image editing, robotics, automated systems, educational tools, and assistive technology for visually impaired people.
